# Работа с токенами темы через MCP

## Жёсткое правило: MCP обязателен

Перед любой работой с токенами темы ОБЯЗАТЕЛЬНО проверь доступность MCP `theme-docs`:

```
CallMcpTool → server: "theme-docs" → toolName: "list_theme_token_groups"
```

**Если MCP недоступен (ошибка, таймаут, отсутствует):**
- НЕМЕДЛЕННО прекрати использование токенов и утилитарных классов по памяти
- Сообщи пользователю: _"MCP theme-docs недоступен. Работа с токенами темы невозможна. Включи MCP `theme-docs` в настройках Cursor и повтори запрос."_
- НЕ используй токены по памяти или из любого другого источника
- НЕ делай исключений — это правило без компромиссов

## Источник информации о дизайн-системе

Для получения информации о дизайн-системе (токены, утилитарные классы, CSS-переменные, типографика, отступы, скругления, цвета) — используй **ТОЛЬКО** MCP `theme-docs`. Никакие другие источники не являются допустимыми.

## Инструменты MCP

MCP-сервер `theme-docs` предоставляет три инструмента:

### 1. `list_theme_token_groups`
Получить список всех групп токенов с кратким описанием. Используй как **первый шаг**.

```json
{ "server": "theme-docs", "toolName": "list_theme_token_groups", "arguments": {} }
```

### 2. `search_theme_tokens`
Найти группу токенов по ключевому слову (назначению или типу).

```json
{ "server": "theme-docs", "toolName": "search_theme_tokens", "arguments": { "query": "отступ" } }
```

### 3. `get_theme_token_group_doc`
Получить полную документацию по группе: CSS-переменные, утилитарные классы, Figma-маппинг, примеры.

```json
{ "server": "theme-docs", "toolName": "get_theme_token_group_doc", "arguments": { "name": "spacing" } }
```

## Рабочий процесс

При необходимости использовать цвет, отступ, типографику или скругление:

1. **Проверь MCP** — вызови `list_theme_token_groups`, убедись что работает
2. **Найди нужную группу** — через `search_theme_tokens` если не знаешь точного имени группы
3. **Прочитай документацию группы** — вызови `get_theme_token_group_doc` для нужной группы
4. **Используй токены** — через утилитарные className или `var(--*)` в inline styles
